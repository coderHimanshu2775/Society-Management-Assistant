Society Management Assistant (SMA) is a comprehensive platform designed to simplify and enhance the management of residential and commercial communities. From security to communication, billing, and maintenance, SMA provides everything needed for efficient community living.

## Features ✨
- **Dashboard** 📊 - Get an overview of society activities in real-time.
- **Communication** 📩 - Send & receive important notices and messages.
- **Marketplace** 🏪 - Buy, sell, or rent properties within the society.
- **Billing & Accounting** 💰 - Manage maintenance bills and track payments.
- **Security Management** 🔒 - Monitor security logs and visitor entries.
- **Vehicle Management** 🚗 - Maintain records of vehicles in the society.
- **Complaint Management** 📢 - File and track complaints effortlessly.
- **Amenities Management** 🏊 - Book and manage society amenities.
- **Admin Dashboard** 🛠️ - A dedicated panel for administrators to manage all operations.
- **Pet Directory** 🐶 - Keep track of pets registered within the community.

## Technologies Used 💻
- **Frontend:** React.js, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Hosting:** Netlify (Frontend), Vercel/Heroku (Backend)
